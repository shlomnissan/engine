\doxysubsection{Detailed Description}
A material for drawing geometries in a flat-\/shaded mode. 

\doxylink{classengine_1_1_flat_material}{Flat\+Material} renders objects without the influence of lighting. The material can be assigned a solid color and optionally a texture map to apply onto the surface. This makes it ideal for simple, unlit geometries.
